Welcome to MeeBible!

MeeBible is the Bible reader for MeeGo Harmattan.

Currently MeeBible supports following translations:

  • New World Translation in 40 different languages
  • Russian Synodal translation

Please start with selecting your language using application menu.

Bible chapters will be downloaded when you open them. Downloaded chapters will be cached and can be read offline later.

You can download whole Bible text at once using “Download Bible” menu. This may take 10‒15 minutes and you'd better do it via WiFi connection. You also have to download full Bible text if you want to use full-text search feature.

Now, please select Bible verse using leftmost button in toolbar and start reading the word of God!
